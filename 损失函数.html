损失函数

在训练机器学习模型时，我们经常会听到‘损失函数’（Loss Function）这个词，比如线性回归用‘均方误差’，逻辑回归用‘交叉熵’。我们为什么要选择这些特定的函数？

本文将深入探讨这些问题，揭示从最大似然估计到最大后验估计的理论框架，是如何自然而然地推导出这些我们耳熟能详的损失函数与正则化项的。

在深入讨论某一个单独的函数之前，我们先来了解一下它的分类。

一、分类

  • 损失函数
    • 最大似然估计 (MLE) - 频率派视角
      • 均方误差MSE：当假设似然函数为高斯分布时导出，等价于最小二乘法。
      • 交叉熵损失：当假设似然函数为伯努利/多项式分布时导出。
    • 最大后验估计 (MAP) - 贝叶斯派视角
      • MAP = MLE + 正则化
      • L2正则化：当参数的先验为高斯分布时导出。
      • L1正则化：当参数的先验为拉普拉斯分布时导出。

主要分为最大似然估计和最大后验估计两派，我们学过的均方误差MSE、交叉熵损失、L1 L2正则化都可以归为里面的一类。

二、最大似然估计

2.1先来了解一些概念

  • 1. 似然（Likelihood） vs. 概率（Probability）
    这是一个关键的区别，但很容易混淆。

    • 概率：是在参数已知的情况下，预测某个结果发生的可能性。

    例如：给定一个均匀的硬币（参数 p=0.5p=0.5p=0.5），抛10次出现7正3反的概率是多少？

    • 似然：是在结果已知的情况下，反推某个参数的可能性。

    例如：已经观测到7正3反（结果已知），硬币是均匀的（参数 p=0.5p=0.5p=0.5）这个说法的似然程度有多大？

    值得注意的是，从数学表达式上看，似然函数 L(θ∣x)L(\theta | x)L(θ∣x) 和概率 P(x∣θ)P(x | \theta)P(x∣θ) 是相等的。它们真正的区别在于看待问题的角度：概率是固定参数 θ\thetaθ 讨论数据 xxx 的分布，而似然则是固定数据 xxx 讨论参数 θ\thetaθ 的可能性。函数本身不变，变量和常量的角色发生了互换。

2.2 什么是最大似然估计 (Maximum Likelihood Estimation)？

最大似然估计的核心思想是：何种参数，最可能产生我们观测到的这批数据？

我们选择的参数，应该是能让现有数据出现的 **似然（Likelihood）**最大的那个。
换言之，我们选择的参数应该让已发生的事件看起来最“理所当然”。

对于独立同分布的样本 X={x1,x2,...,xn}X = \{x_1, x_2, ..., x_n\}X={x1​,x2​,...,xn​}，其似然函数为：
L(θ∣X)=P(X∣θ)=∏i=1nP(xi∣θ)L(\theta | X) = P(X|\theta) = \prod_{i=1}^{n} P(x_i|\theta)L(θ∣X)=P(X∣θ)=i=1∏n​P(xi​∣θ)

要取最大似然函数，也就是把 P(X∣θ)P(X|\theta)P(X∣θ) ，取到最大值。
即在θ\thetaθ条件下把产生观测到的这批数据的概率取到最大。

由于对似然函数直接求导和优化通常很复杂（尤其是连乘形式），我们通常将其转换为对数似然函数。最大化似然函数等价于最大化对数似然（因为对数在低数大于1的时候时单调递增的），而最大化对数似然则等价于最小化负对数似然（就是增函数取负变成减函数）。

最小化负对数似然
θML=arg⁡min⁡θ∑i=1m−log⁡pmodel(x(i)∣θ) \theta_{ML} = \arg \min_{\theta} \sum_{i=1}^{m} -\log p_{model}(x^{(i)} | \theta) θML​=argθmin​i=1∑m​−logpmodel​(x(i)∣θ)

2.3 最大似然估计和均方误差有什么关系？

先说结论：在线性回归模型中，假设误差服从均值为0的正态分布，那么通过最小化均方误差来求解模型参数，与通过最大似然估计来求解模型参数是等价的。

均方误差是什么？
均方误差是衡量预测值与真实值之间差异的一种常用指标。 它计算的是预测误差平方的平均值。 对于一组真实值为 y={y1,y2,...,yn}y = \{y_1, y_2, ..., y_n\}y={y1​,y2​,...,yn​}，预测值为 y^={y^1,y^2,...,y^n}\hat{y} = \{\hat{y}_1, \hat{y}_2, ..., \hat{y}_n\}y^​={y^​1​,y^​2​,...,y^​n​} 的样本，其均方误差定义为：
MSE=1n∑i=1n(yi−y^i)2MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2MSE=n1​i=1∑n​(yi​−y^​i​)2
MSE的值越小，说明模型的预测越精确。

最大似然估计和均方误差之间的关系，在线性回归模型中得到了完美的诠释。 假设一个线性回归模型：
y=f(x;θ)+ϵy = f(x; \theta) + \epsilony=f(x;θ)+ϵ
其中，yyy 是因变量，xxx 是自变量，θ\thetaθ 是模型参数，ϵ\epsilonϵ 是误差项。

关键假设：误差服从正态分布
为了将最大似然估计与均方误差联系起来，我们需要一个关键的假设：误差项 ϵ\epsilonϵ 服从均值为0，方差为 σ2\sigma^2σ2 的正态分布（高斯分布）。 这个假设在很多实际应用中是合理的，因为根据中心极限定理，大量独立的随机因素叠加产生的误差往往近似于正态分布。

基于这个假设，对于给定的输入 xix_ixi​，其对应的输出 yiy_iyi​ 也服从正态分布：
yi∼N(f(xi;θ),σ2)y_i \sim N(f(x_i; \theta), \sigma^2)yi​∼N(f(xi​;θ),σ2)

在xix_ixi​确定之后，把 f(xi;θ)f(x_i; \theta)f(xi​;θ)看成常数，因为y=f(x;θ)+ϵy = f(x; \theta) + \epsilony=f(x;θ)+ϵ，又因为误差项 ϵ\epsilonϵ 服从均值为0，方差为 σ2\sigma^2σ2 的正态分布，所以yiy_iyi​ 服从均值为f(xi;θ)f(x_i; \theta)f(xi​;θ)，方差为 σ2\sigma^2σ2 （加减常数不改变方差）的正态分布。

其概率密度函数为：
P(yi∣xi;θ,σ2)=12πσ2exp⁡(−(yi−f(xi;θ))22σ2)P(y_i | x_i; \theta, \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(y_i - f(x_i; \theta))^2}{2\sigma^2}\right)P(yi​∣xi​;θ,σ2)=2πσ2​1​exp(−2σ2(yi​−f(xi​;θ))2​)

接下来，我们就可以构建似然函数了。

  1. 构建似然函数：
    对于整个数据集，假设样本之间相互独立，其似然函数为所有样本概率密度函数的乘积：
    L(θ,σ2∣Y,X)=∏i=1nP(yi∣xi;θ,σ2)=∏i=1n12πσ2exp⁡(−(yi−f(xi;θ))22σ2)L(\theta, \sigma^2 | Y, X) = \prod_{i=1}^{n} P(y_i | x_i; \theta, \sigma^2) = \prod_{i=1}^{n} \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(y_i - f(x_i; \theta))^2}{2\sigma^2}\right)L(θ,σ2∣Y,X)=i=1∏n​P(yi​∣xi​;θ,σ2)=i=1∏n​2πσ2​1​exp(−2σ2(yi​−f(xi​;θ))2​)
    上面我们知道，我们通常将似然函数转换为对数似然函数，于是得到：
    ln⁡L(θ,σ2∣Y,X)=∑i=1nln⁡(12πσ2exp⁡(−(yi−f(xi;θ))22σ2))\ln L(\theta, \sigma^2 | Y, X) = \sum_{i=1}^{n} \ln \left( \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(y_i - f(x_i; \theta))^2}{2\sigma^2}\right) \right)lnL(θ,σ2∣Y,X)=i=1∑n​ln(2πσ2​1​exp(−2σ2(yi​−f(xi​;θ))2​))

这里log的底数可以取任意大于1的值，都是单调递增的。为了方便，我们不妨取e。

我们的目标是找到使对数似然函数最大化的参数 θ\thetaθ。观察上式，第一项 −n2ln⁡(2πσ2)-\frac{n}{2}\ln(2\pi\sigma^2)−2n​ln(2πσ2) 是一个与 θ\thetaθ 无关的常数。 因此，最大化对数似然函数就等价于最小化下式：
∑i=1n(yi−f(xi;θ))2\sum_{i=1}^{n} (y_i - f(x_i; \theta))^2i=1∑n​(yi​−f(xi​;θ))2

我们惊讶地发现：最小化上式等价于最小化均方误差 ！
MSE=1n∑i=1n(yi−y^i)2MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2MSE=n1​i=1∑n​(yi​−y^​i​)2
其中 y^i=f(xi;θ)\hat{y}_i = f(x_i; \theta)y^​i​=f(xi​;θ)。

因此，我们得出了一个重要的结论：在线性回归模型中，假设误差服从均值为0的正态分布，那么通过最小化均方误差来求解模型参数，与通过最大似然估计来求解模型参数是等价的。 这也解释了为什么在线性回归中，我们通常使用最小二乘法（其目标函数就是均方误差）来拟合模型。

2.4 最大似然估计和交叉熵有什么关系？

先上结论： 对模型进行最大似然估计，就等同于最小化 模型预测的概率分布 与 训练数据的经验分布 之间的交叉熵。
很多名词，我们来一个个解释：

模型预测的概率分布，就是我们上面提到的 模型预测“概率密度函数”。

训练数据的经验分布:我们可以用这个有限的数据集来定义一个经验分布 (Empirical Distribution) p^data(x)\hat{p}_{data}(x)p^​data​(x)。
对于任何一个值 xxx，它在数据集 DDD 中出现的频率就是它的经验概率。如果数据集中有 mmm 个样本，其中值 xxx 出现了 count(x)count(x)count(x) 次，那么：
p^data(x)=count(x)m \hat{p}_{data}(x) = \frac{\text{count}(x)}{m} p^​data​(x)=mcount(x)​
这个经验分布 p^data\hat{p}_{data}p^​data​ 是我们基于观测数据对真实数据分布 pdatap_{data}pdata​ 的一个近似。

交叉熵（Cross-Entropy）是用来衡量两个概率分布之间差异的指标。对于离散分布 PPP 和 QQQ，其交叉熵定义为：
H(P,Q)=−∑xP(x)log⁡Q(x) H(P, Q) = - \sum_{x} P(x) \log Q(x) H(P,Q)=−x∑​P(x)logQ(x)

了解了这些，我们终于可以开始推导了。
最大化对数似然函数与最大化平均对数似然函数（就是多除了一个m）是等价的，因为 1/m1/m1/m 是一个正的常数，不影响 arg⁡max⁡\arg \maxargmax 的取值。
θML=arg⁡max⁡θ1m∑i=1mlog⁡pmodel(xi∣θ) \theta_{ML} = \arg \max_{\theta} \frac{1}{m} \sum_{i=1}^{m} \log p_{model}(x_i | \theta) θML​=argθmax​m1​i=1∑m​logpmodel​(xi​∣θ)
接下来，我们可以对求和项进行重新组合。我们不再对数据集中的每一个样本 xix_ixi​ 进行求和，而是对所有可能的数据值 xxx 进行求和，并用它们在数据集中出现的次数进行加权：
1m∑i=1mlog⁡pmodel(xi∣θ)=1m∑xcount(x)log⁡pmodel(x∣θ) \frac{1}{m} \sum_{i=1}^{m} \log p_{model}(x_i | \theta) = \frac{1}{m} \sum_{x} \text{count}(x) \log p_{model}(x | \theta) m1​i=1∑m​logpmodel​(xi​∣θ)=m1​x∑​count(x)logpmodel​(x∣θ)
现在，把 1m\frac{1}{m}m1​ 乘到求和符号里面：
=∑xcount(x)mlog⁡pmodel(x∣θ) = \sum_{x} \frac{\text{count}(x)}{m} \log p_{model}(x | \theta) =x∑​mcount(x)​logpmodel​(x∣θ)
我们发现，count(x)m\frac{\text{count}(x)}{m}mcount(x)​ 正是经验分布 p^data(x)\hat{p}_{data}(x)p^​data​(x) 的定义！所以上式可以写为：
=∑xp^data(x)log⁡pmodel(x∣θ) = \sum_{x} \hat{p}_{data}(x) \log p_{model}(x | \theta) =x∑​p^​data​(x)logpmodel​(x∣θ)
因此，我们证明了：
arg⁡max⁡θ∑i=1mlog⁡pmodel(xi∣θ)⟺arg⁡max⁡θ∑xp^data(x)log⁡pmodel(x∣θ) \arg \max_{\theta} \sum_{i=1}^{m} \log p_{model}(x_i | \theta) \quad \Longleftrightarrow \quad \arg \max_{\theta} \sum_{x} \hat{p}_{data}(x) \log p_{model}(x | \theta) argθmax​i=1∑m​logpmodel​(xi​∣θ)⟺argθmax​x∑​p^​data​(x)logpmodel​(x∣θ)
两者是完全等价的。
因此，我们得到结论： 对模型进行最大似然估计，就等同于最小化 模型预测的概率分布 与 训练数据的经验分布 之间的交叉熵。

拓展：为什么选择它作为损失函数？
在神经网络的训练中，我们依赖梯度下降来更新参数。损失函数对模型输出的梯度（导数）至关重要。
当交叉熵损失与Softmax激活函数（在多分类任务中是标配）结合时，它会产生一个非常简洁和高效的梯度。有兴趣可以自行查阅。

3.1 最大后验估计是什么

贝叶斯派的核心不同在于：它认为参数 θ\thetaθ 不是一个固定的常量，而是一个随机变量。在我们观测数据之前，我们对 θ\thetaθ 就有一个先验信念（Prior Belief），这个信念由一个先验分布 f(θ)f(\theta)f(θ) 来描述。

目标: arg⁡max⁡θP(Y,X∣θ)f(θ)\arg\max_{\theta} P(Y, X | θ) f(θ)argmaxθ​P(Y,X∣θ)f(θ)

  • 贝叶斯派的目标是最大化后验概率（Posterior Probability） P(θ∣Y,X)P(\theta | Y, X)P(θ∣Y,X)。根据贝叶斯定理：
    P(θ∣Y,X)=P(Y,X∣θ)⋅f(θ)P(Y,X)P(\theta | Y, X) = \frac{P(Y, X | \theta) \cdot f(\theta)}{P(Y, X)}P(θ∣Y,X)=P(Y,X)P(Y,X∣θ)⋅f(θ)​
    (后验概率 = (似然 × 先验) / 证据)
    *在最大化这个式子时，分母 P(Y,X)P(Y,X)P(Y,X) 与 θ\thetaθ 无关，可以忽略。因此，目标就变成了最大化分子部分：似然 × 先验，即 P(Y, X | θ) f(θ)。
  • 核心思想（最大后验估计, MAP）：结合我们对数据的观测（似然）和我们对参数的先验信念（先验），找到最可信的参数 θ\thetaθ。

推导过程:

  • 与MLE类似，我们对目标函数 P(Y,X∣θ)f(θ)P(Y, X | θ) f(θ)P(Y,X∣θ)f(θ) 取对数，然后取负，将最大化问题转为最小化问题。
    原式：argminθ[−log(P(Y,X∣θ)f(θ))]argmin_θ [ -log(P(Y, X | θ) f(θ)) ]argminθ​[−log(P(Y,X∣θ)f(θ))]
    ⟹\Longrightarrow⟹ argminθ[−log(P(Y,X∣θ))−log(f(θ))]argmin_θ [ -log(P(Y, X | θ)) - log(f(θ)) ]argminθ​[−log(P(Y,X∣θ))−log(f(θ))]
  • 我们已经从左侧知道，第一项 −log(P(Y,X∣θ))-log(P(Y, X | θ))−log(P(Y,X∣θ)) 对应着最小二乘法的损失项 ∑(yi−f(xi;θ))2\sum (y_i - f(x_i; \theta))^2∑(yi​−f(xi​;θ))2。
    结论: argmin_θ [ Σ(y_i - f(x_i; θ))^2 - log(f(θ)) ]
  • 这个最终的损失函数由两部分构成：
    • 第一部分：和MLE一样的最小二乘损失项。
    • 第二部分: -log(f(θ))，这一项来自于我们对参数的先验分布。它在机器学习中扮演着**正则化项（Regularization Term）**的角色。

例如，如果我们假设参数的先验分布 f(θ)f(\theta)f(θ) 是一个高斯分布，那么 -log(f(θ)) 就会变成一个 L2正则化项（Ridge回归）。如果假设是拉普拉斯分布，它就会变成一个 L1正则化项（LASSO回归）。

2. 拉普拉斯先验 (Laplace Prior) → L1 正则化 (LASSO)

第一步：写出拉普拉斯先验的概率密度函数

现在，我们换一个先验信念：我们认为参数 θ\thetaθ 应该是稀疏的，即很多 θj\theta_jθj​ 应该正好是0。这个信念可以用拉普拉斯分布来很好地描述，因为它在0点有一个尖峰。

单个参数 θj\theta_jθj​ 服从均值为0、尺度参数为bbb的拉普拉斯分布：
f(θj)=12bexp⁡(−∣θj∣b)f(\theta_j) = \frac{1}{2b} \exp\left(-\frac{|\theta_j|}{b}\right)f(θj​)=2b1​exp(−b∣θj​∣​)

同样，假设独立，整个参数向量 θ\thetaθ 的先验概率为：
f(θ)=∏j=1n12bexp⁡(−∣θj∣b)=(12b)nexp⁡(−1b∑j=1n∣θj∣)f(\theta) = \prod_{j=1}^{n} \frac{1}{2b} \exp\left(-\frac{|\theta_j|}{b}\right) = \left(\frac{1}{2b}\right)^n \exp\left(-\frac{1}{b}\sum_{j=1}^{n}|\theta_j|\right)f(θ)=∏j=1n​2b1​exp(−b∣θj​∣​)=(2b1​)nexp(−b1​∑j=1n​∣θj​∣)

第二步：计算 −log⁡(f(θ))-\log(f(\theta))−log(f(θ))

同样取负对数：
−log⁡(f(θ))=−log⁡[(12b)nexp⁡(−1b∑j=1n∣θj∣)]-\log(f(\theta)) = -\log\left[ \left(\frac{1}{2b}\right)^n \exp\left(-\frac{1}{b}\sum_{j=1}^{n}|\theta_j|\right) \right]−log(f(θ))=−log[(2b1​)nexp(−b1​∑j=1n​∣θj​∣)]
=−[nlog⁡(12b)−1b∑j=1n∣θj∣]= -\left[ n\log\left(\frac{1}{2b}\right) - \frac{1}{b}\sum_{j=1}^{n}|\theta_j| \right]=−[nlog(2b1​)−b1​∑j=1n​∣θj​∣]
=1b∑j=1n∣θj∣−nlog⁡(12b)= \frac{1}{b}\sum_{j=1}^{n}|\theta_j| - n\log\left(\frac{1}{2b}\right)=b1​∑j=1n​∣θj​∣−nlog(2b1​)

第三步：放入优化目标中

代入并舍去与 θ\thetaθ 无关的常数项，优化目标变为：
argminθ[Original_Loss+1b∑j=1n∣θj∣]\text{argmin}_{\theta} \left[ \text{Original\_Loss} + \frac{1}{b}\sum_{j=1}^{n}|\theta_j| \right]argminθ​[Original_Loss+b1​∑j=1n​∣θj​∣]

如果我们令正则化系数 λ=1b\lambda = \frac{1}{b}λ=b1​，就得到了LASSO回归的损失函数：
argminθ[Original_Loss+λ∑j=1n∣θj∣]\text{argmin}_{\theta} \left[ \text{Original\_Loss} + \lambda \sum_{j=1}^{n}|\theta_j| \right]argminθ​[Original_Loss+λ∑j=1n​∣θj​∣]

这里的 ∑∣θj∣\sum |\theta_j|∑∣θj​∣ 就是L1正则化项。

四、总结

至此，我们终于把损失函数讲完了。

我们从最大似然估计 (MLE)出发，到最大后验估计 (MAP)，两者关系不过就是MAP损失 = MLE损失 + 参数先验的负对数。

而最大后验估计和L1、L2的关系：
最大后验估计 + 参数服从拉普拉斯分布的先验   ⟹  \implies⟹ L1 正则化
最大后验估计 + 参数服从高斯分布的先验   ⟹  \implies⟹ L2 正则化
